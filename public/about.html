About SpareTime

SpareTime helps you get more done in short pockets of time. It suggests a few quick, prioritized tasks you can finish with the time you have, helps organize chores and projects, and keeps your task data available when you are offline. You can install SpareTime on your phone or desktop for an app-like experience.

Key details

  • Type: Mobile-first web app (Progressive Web App)
  • Who it's for: People who want quick, practical task suggestions and lightweight task management
  • Optional backup: Two-way backup to Google Drive's AppData folder (opt-in)

Google Drive access

When you choose to enable cloud backup, SpareTime uses the Google Drive API with the following scope:

https://www.googleapis.com/auth/drive.appdata

SpareTime only gets permission to read and write files that it creates in the signed-in user's Drive AppData folder. It does not access, list, or modify any other files in the user's Google Drive.
